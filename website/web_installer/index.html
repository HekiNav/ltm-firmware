Hekinav Live Train Map Setup

Wi-Fi setup and updates for your Live Train Map.

  1. Plug your map into a USB port on your computer using a usb cable.
    Note: For setup you can only use the USB port on the left side on the Helsinki map.
  2. Make sure you have selected the correct board below.
  3. Click "Connect" and select the correct COM port.
    Look for the COM port labeled as "USB JTAG/serial debug unit".
  4. Select "Connect to Wi-Fi" and follow the prompts.
    Installing the latest Firmware is optional.
